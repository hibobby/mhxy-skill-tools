# 生成 Tauri 应用图标

## 快速方法（推荐）

### 使用 Tauri CLI 自动生成

1. **确保已安装 Tauri CLI**
   ```bash
   npm install -g @tauri-apps/cli
   ```

2. **准备一个源图片**
   - 格式：PNG
   - 尺寸：至少 512x512 像素（越大越好）
   - 内容：你的应用图标

3. **生成所有图标**
   ```bash
   cd src-tauri
   tauri icon path/to/your/icon.png
   ```
   
   这会在 `src-tauri/icons/` 目录下自动生成所有需要的图标文件。

4. **完成后恢复配置**
   编辑 `src-tauri/tauri.conf.json`，将：
   ```json
   "bundle": {
     "active": false,
     "targets": "all"
   }
   ```
   改回：
   ```json
   "bundle": {
     "active": true,
     "targets": "all",
     "icon": [
       "icons/32x32.png",
       "icons/128x128.png",
       "icons/128x128@2x.png",
       "icons/icon.icns",
       "icons/icon.ico"
     ]
   }
   ```

## 手动创建方法

### Windows 图标 (.ico)

1. 使用在线工具：
   - https://www.icoconverter.com/
   - 上传 PNG 图片，选择多尺寸导出
   - 下载 .ico 文件

2. 使用工具软件：
   - Greenfish Icon Editor
   - IcoFX
   - GIMP (需要插件)

### macOS 图标 (.icns)

1. 使用在线工具：
   - https://cloudconvert.com/png-to-icns

2. 使用命令行（macOS）：
   ```bash
   # 创建图标集
   iconutil -c icns icon.iconset
   ```

### PNG 图标

使用任意图片编辑软件：
- 创建 32x32、128x128、256x256 的 PNG 图片
- 保存到 `src-tauri/icons/` 目录

## 占位图标（仅用于开发）

如果只是开发和测试，可以：

1. **使用简单纯色图标**
   - 创建一个 512x512 的纯色 PNG（例如蓝色方块）
   - 使用在线工具转换为 .ico
   - 或者暂时禁用 bundle

2. **临时禁用图标要求**
   已在 `tauri.conf.json` 中设置 `"active": false`，开发时可以暂时不提供图标。

## 注意事项

- 开发时（`npm run tauri dev`）：不需要图标
- 打包时（`npm run tauri build`）：需要所有图标文件
- 图标文件建议使用高质量的原图（至少 512x512）
